Wild Animals & Their Habitats

About

Wild Animals & Their Habitats is a small educational web app that teaches you about six different animals from around the world and the environments they live in.

Features

  • Pick an animal from a dynamic menu to learn about it
  • See a picture, the habitat, and a short description
  • Tap the Listen button to hear the description read aloud
  • Works offline after the first visit
  • Can be installed to your device like a real app

Try It

Open the web app

How to Install

  1. Open the app in Chrome or Edge.
  2. Click the Install button in the top-right corner of the app.
  3. Or use the install icon in the browser's address bar.
  4. The app will be added to your device and can run offline.
