User Documentation

What the app does

This app shows information about wild animals and the habitats they live in. Each animal has a picture, a description, and an audio button that reads the description aloud.

How to use it

  1. Open Animal Page.html in your browser.
  2. You will see a green menu of animals below the title.
  3. Click any animal name to see its picture, habitat, and description.
  4. Click the Listen button to hear the description spoken aloud.

How to install the app

On Chrome or Edge, an Install button appears in the top-right corner when installation is available. Click it and confirm.

You can also install from the browser's address bar by looking for the install icon.

Note: install only works when the app is served over http:// or https:// (for example, with a local web server). Just double-clicking the HTML file uses a file:// URL where install and service workers are not available.

Offline use

The first time you visit the app, the service worker saves its files. After that, you can open it without an internet connection.

Browser support

The app works best on recent versions of Chrome, Edge, and Firefox.
